About

Saya Nurul Arifin, seorang mahasiswa. Kecintaan saya pada dunia komputer mengantarkan saya pada pendidikan IT. Saya lulusan SMK Jurusan TKJ di SMKN 1 Gombong (2006-2009). Setelah sekian tahun lulus, akhirnya pada tahun 2019 saya mengambil pendidikan lanjut di AMIK PGRI Kebumen dengan jurusan Manajemen Informatika.

Saat ini saya bekerja sebagai Teknisi di Instansi Pendidikan di Kabupaten Kebumen, sejak Tahun 2011.

Website ini membagikan profil singkat saya dan membagikan sedikit pengalaman saya. Silakan kontak saya jika ingin berkorespondensi.
